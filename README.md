Sam's Awesome WM config
=======================


1) Required Utils
-----------------

- Saidar
    #pacman -S libstatgrab pystatgrab

- Dfc (disk util)
Can be found on Arch's AUR site

- Alsa Utils (Amix) 
	#pacman -S alsa-utils

- Powertop (temperature)
	#pacman -S powertop

- HDDTemp
	#pacman -S hddtemp

2) Required Fonts
-----------------

- Termsyn
Can be found on Arch's AUR site

- Terminus
Official package repo
	#pacman -S terminus-font
